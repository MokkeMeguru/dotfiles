#+TITLE: Dotfiles

* How to use

#+begin_src shell
cd ~
git clone https://github.com/MokkeMeguru/dotfiles
cd dotfiles
chmod +x install.sh
./install.sh --help
#+end_src

And then, you need to follow the instruction.

* Example
- osx

  #+begin_src shell
./install -o osx -l
  #+end_src

- archlinux / manjaro linux

  #+begin_src shell
./install -o arch -l
  #+end_src
