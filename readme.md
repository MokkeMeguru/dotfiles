# Dotfiles

## How to use

``` shell
cd ~
git clone https://github.com/MokkeMeguru/dotfiles
cd dotfiles
chmod +x install.sh
./install.sh --help
```

And then, you need to follow the instruction.

### Example

- osx

``` shell
./install.sh -o osx -l
```

- archlinux / manjaro linux

``` shell
./install.sh -o arch -l
```
